# 大厅任务循环

## 模块定位

承接玩家登录后的第一目标，负责把玩家从大厅导向短任务、奖励领取和工坊升级。

## 系统规则

玩家每天获得三次基础任务机会。完成任务后获得星尘、零件和少量工坊经验。

## 资源产出/消耗

产出星尘、零件和工坊经验；消耗任务次数和少量现实等待时间。

## 关联截图

- @img_lobby_overview: 大厅总览示意

## 玩家主要操作

查看大厅任务卡，点击开始任务，完成后领取奖励，再根据奖励提示进入工坊。

## 乐趣点

任务完成反馈很短，奖励落袋感清楚，适合首日快速建立循环理解。

## 主要问题

大厅同时出现任务、工坊和活动入口，首屏优先级略拥挤。

## 优化方向

首日仅高亮任务入口和工坊入口，其他入口延后出现或降低视觉权重。
