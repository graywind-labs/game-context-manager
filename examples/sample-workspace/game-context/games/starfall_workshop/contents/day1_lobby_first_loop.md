# 第一天大厅首次任务循环

## 账号状态

- 创角天数：1
- 累计付费金额：0
- 最大主线通关数：1-2
- 角色等级：4

## 关联截图

- @img_lobby_overview: 大厅总览示意

## 过程说明

玩家登录后先看到 @img_lobby_overview，大厅中央任务卡提示可开始短任务。完成第一轮任务后，奖励弹窗引导玩家进入工坊查看升级。

## 乐趣点（主观）

第一次任务链路短，玩家能在一分钟内看到任务完成、奖励发放和工坊入口。

## 已知问题（主观）

任务卡和活动入口距离较近，未读文案时可能误点到活动入口。

## 优化方向（主观）

首轮任务完成前隐藏活动入口，或将活动入口视觉权重降低。
