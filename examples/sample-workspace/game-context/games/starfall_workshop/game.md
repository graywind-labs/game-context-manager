# 星坠工坊

## 项目阶段

测试

## 游戏类型

轻度放置 + 工坊经营。

## 核心玩法

玩家从大厅接取短任务，获得星尘与零件，再回到工坊升级设备并解锁更高收益任务。

## 主要乐趣

短周期任务反馈清楚，工坊升级能让数值成长被直接感知。

## 目标用户

偏好低压力养成、轻策略资源规划和每日短会话的玩家。

## 当前运营目标

验证前两天任务循环是否足够清晰，并观察升级材料卡点是否过早出现。

## 当前主要问题

大厅任务入口与工坊升级入口的优先级不够明确，新玩家可能不知道下一步应先做任务还是先升级。

## 主要优化方向

强化首日引导中的下一步提示，并在升级材料不足时给出可执行的获取路径。
